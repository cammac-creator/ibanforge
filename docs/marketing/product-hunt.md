# Product Hunt Launch

## Tagline (60 chars max)

Free IBAN validation & BIC lookup API for AI agents

## Description

IBANforge validates IBANs and looks up BIC/SWIFT codes for 80+ countries. Built for developers and AI agents.

- 39,000+ BIC entries from GLEIF (open data)
- MCP integration for Claude, GPT, and other AI agents
- Free during launch -- no API key, no subscription
- Interactive playground to test instantly
- Full documentation with curl, Python, and TypeScript examples
- Self-hostable (MIT license, Docker)

Perfect for: payment processing, KYC compliance, invoice automation, and AI agent workflows.

Hey Product Hunt!
