# Contributing to IBANforge

We welcome contributions! Here's how you can help:

## Reporting Bugs
Open an issue on GitHub with:
- What you expected
- What happened
- Steps to reproduce

## Adding BIC Data
We're always looking to expand our BIC database (currently 39K entries from GLEIF). If you have access to open/free BIC data sources, please open an issue.

## Code Contributions
1. Fork the repo
2. Create a feature branch
3. Make your changes
4. Run tests: `npm test`
5. Submit a PR

## Data Sources We're Looking For
- National bank registries (Bundesbank, SIX Group, etc.)
- Open banking directories
- Community-maintained BIC databases (with compatible licenses)
