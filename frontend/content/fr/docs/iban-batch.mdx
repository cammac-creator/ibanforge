# Validation par lot

Validez jusqu'a 100 IBAN en un seul appel API. Chaque IBAN est traite independamment — un IBAN invalide n'affecte pas les autres. 60 % moins cher que les appels individuels.

## Endpoint

```
POST https://api.ibanforge.com/v1/iban/batch
```

**Cout :** $0.002 USDC par IBAN (ex. : 10 IBAN = $0.020, 100 IBAN = $0.200)

## Requete

### En-tetes

| En-tete | Valeur | Requis |
|---|---|---|
| `Content-Type` | `application/json` | Oui |
| `X-PAYMENT` | Token de paiement x402 | Oui |

### Corps

```json
{
  "ibans": [
    "CH93 0076 2011 6238 5295 7",
    "DE89 3704 0044 0532 0130 00",
    "GB29 NWBK 6016 1331 9268 19",
    "INVALID123"
  ]
}
```

| Champ | Type | Description |
|---|---|---|
| `ibans` | `string[]` | Tableau d'IBAN a valider. De 1 a 100 elements. |

### Champs de la reponse

**Premier niveau :**

| Champ | Type | Description |
|---|---|---|
| `results` | `array` | Tableau des resultats de validation, dans le meme ordre que l'entree |
| `count` | `number` | Nombre total d'IBAN traites |
| `valid_count` | `number` | Nombre d'IBAN valides |
| `cost_usdc` | `number` | Cout total de ce lot |
| `processing_ms` | `number` | Temps total de traitement |

**Chaque element de `results`** possede la meme structure que la [reponse de validation unitaire](/docs/iban-validate), incluant `sepa`, `issuer` et `risk_indicators` pour les IBAN valides.

### Erreurs

| Statut | Code | Description |
|---|---|---|
| 400 | `empty_batch` | Le tableau `ibans` est vide |
| 400 | `batch_too_large` | Plus de 100 IBAN soumis |
| 400 | `invalid_request` | Le corps de la requete n'est pas un JSON valide ou le champ `ibans` est manquant |

## Exemples de code

### cURL

```bash
curl -X POST https://api.ibanforge.com/v1/iban/batch \
  -H "Content-Type: application/json" \
  -d '{
    "ibans": [
      "CH9300762011623852957",
      "DE89370400440532013000",
      "FR7630006000011234567890189"
    ]
  }'
```

### Python

```python
import requests

response = requests.post(
    "https://api.ibanforge.com/v1/iban/batch",
    json={
        "ibans": [
            "CH9300762011623852957",
            "DE89370400440532013000",
            "FR7630006000011234567890189",
        ]
    },
)

data = response.json()
print(f"Valid: {data['valid_count']}/{data['count']}")

for result in data["results"]:
    if result["valid"]:
        risk = result["risk_indicators"]["country_risk"]
        issuer = result["issuer"]["type"]
        print(f"  {result['iban']} — {result['country']['name']} ({issuer}, risk: {risk})")
    else:
        print(f"  INVALID — {result['error_detail']}")
```

```typescript
  }
}
```
