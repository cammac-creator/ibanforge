# IBANforge API

[Postman Collection herunterladen](/ibanforge-postman.json)

IBANforge ist eine schnelle, zuverlässige API für **IBAN-Validierung** und **BIC/SWIFT-Abfrage**. Sie deckt über 80 Länder ab und bietet Echtzeit-Prüfsummenverifizierung, BBAN-Strukturanalyse, angereicherte Institutsdaten von GLEIF, **SEPA-Konformitätsinformationen**, **Emittentenklassifizierung** (Bank vs. E-Geld-Institut/Neobank) und **Risikoindikatoren** für Compliance-Agenten.

## Basis-URL

```
https://api.ibanforge.com
```

Alle Endpunkte werden über HTTPS bereitgestellt. HTTP-Anfragen werden automatisch umgeleitet.

## Authentifizierung

IBANforge verwendet **x402-Mikrozahlungen** anstelle herkömmlicher API-Schlüssel. Jede Anfrage kostet einen Bruchteil eines Cents und wird automatisch in USDC bezahlt. Keine Registrierung, kein Abonnement, keine Ratenlimits.

Ihr HTTP-Client fügt einen Zahlungs-Header hinzu, und die API verifiziert diesen vor der Verarbeitung. Weitere Details finden Sie im [Leitfaden zu x402-Mikrozahlungen](/docs/x402).

## Endpunkte im Überblick

| Endpunkt | Methode | Kosten | Beschreibung |
|---|---|---|---|
| `/v1/iban/validate` | POST | $0.005 | Einzelne IBAN validieren |
| `/v1/iban/batch` | POST | $0.002/IBAN | Bis zu 100 IBANs validieren |
| `/v1/bic/:code` | GET | $0.003 | BIC/SWIFT-Code abfragen |
| `/v1/demo` | GET | Kostenlos | Beispielvalidierungen (keine Zahlung erforderlich) |
| `/health` | GET | Kostenlos | Service-Gesundheitsprüfung |

## Schnellbeispiel

```bash
curl -X POST https://api.ibanforge.com/v1/iban/validate \
  -H "Content-Type: application/json" \
  -d '{"iban": "CH93 0076 2011 6238 5295 7"}'
```

## Kostenloses Testen

Verwenden Sie den Endpunkt `/v1/demo`, um die API ohne Zahlungseinrichtung zu testen:

```bash
curl https://api.ibanforge.com/v1/demo
```

Dieser gibt vorvalidierte Beispiel-IBANs aus mehreren Ländern zurück, damit Sie das Antwortformat erkunden können.
